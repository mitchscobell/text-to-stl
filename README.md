# Text to STL

## BUILD INSTRUCTIONS

```bash
  npm install
  npm run webpack
  cd dist
  # browse to index.html in your web browser or copy to server, for example apache2
  sudo cp -r $HOME/text-to-stl/dist/* /var/www/html
```

OR Running locally

```bash
npm run tsc
node lib/cli.js
```
